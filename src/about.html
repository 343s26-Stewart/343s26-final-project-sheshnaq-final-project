How does this all work?

Under the hood

We pull from two API's. The first of them fetches info from wikipedia that we add onto our country page. This way the user can be informed about their next potential trip.

Our other API fetches various pieces of data on a specific country. We then use that data to fill out fields in our country page as well as our bucket List page.
